# Guia de Estudo de ADVPL

Este guia foi criado para auxiliar no estudo de ADVPL (Advanced Protheus Language), uma linguagem de programação utilizada no desenvolvimento de customizações para o ERP Protheus da TOTVS.

## Introdução

ADVPL é uma linguagem de programação procedural baseada em xBase, amplamente utilizada para personalização e extensão do ERP Protheus. Este guia aborda conceitos básicos, estruturas de programação e melhores práticas para desenvolvimento em ADVPL.

## Conteúdos

### 1. Fundamentos de ADVPL

- **Estrutura de um programa ADVPL**
  - Estrutura básica do código ADVPL.
  - Definição de funções e procedimentos.
  - Variáveis e tipos de dados em ADVPL.

- **Operadores e Expressões**
  - Operadores aritméticos, lógicos e relacionais.
  - Manipulação de strings e datas.

### 2. Estruturas de Controle

- **Condicionais**
  - Uso de `IF`, `ELSEIF` e `ENDIF`.
  - Operadores condicionais (`AND`, `OR`).

- **Laços de Repetição**
  - Uso de `DO WHILE`, `DO CASE` e `FOR`.

### 3. Banco de Dados

- **Manipulação de Tabelas**
  - Acesso e manipulação de dados em tabelas do Protheus.
  - Uso de comandos `DBSELECT`, `DBUPDATE`, `DBDELETE`, etc.

### 4. Funções Avançadas

- **Funções Nativas**
  - Utilização de funções padrão do Protheus.
  - Desenvolvimento de funções customizadas.

### 5. Boas Práticas e Dicas

- **Organização de Código**
  - Convenções de nomenclatura.
  - Comentários e documentação de código.

- **Debugging**
  - Uso de debuggers e técnicas de depuração.
